This file contains the data output received from Ken after variants from GnomAD were formatted.
Files hgvsg_corrected.csv and variants-hg19.csv were input files sent to Ken.
Files anno_gn.tsv and anno_myv.tsv contains the output information from Ken. anno_gn contains information from OncoKB database for the variants while anno_myv contains variant information aggregated through MyVariant.
